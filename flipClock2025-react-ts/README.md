# 2025新春版：干货满满!如何优雅简洁地实现时钟翻牌器(支持JS/Vue/React)
## React18/19 Typescript版

> 本项目提供的是组件源码，并不是npm依赖包。适合直接放到实际项目中使用，可根据实际项目需求自行修改。

## 使用方法

安装依赖包：
```
npm install
```

运行项目：
```
npm run dev
```

build项目：
```
npm run build
```
